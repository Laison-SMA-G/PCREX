PCREX - Electron Admin Store (no-login)

Features:
- Electron + React (Vite) + Tailwind
- Backend: Express + MongoDB (database name: Rex)
- No login: app opens directly to Dashboard
- Sales line chart on Sales page
- Image uploads for products (stored in server/uploads)

Run:
1. Unzip and open the project root.
2. Install root deps: npm install
3. Install server deps: cd server && npm install
4. Install frontend deps: cd ../frontend && npm install
5. Start backend (keep this terminal open): cd ../server && npm start
6. In project root, start Electron: npm start

Notes:
- Backend default port: 5175
- Frontend dev server: 5173
